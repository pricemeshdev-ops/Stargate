# STARGATE Report Archive

This directory stores the local STARGATE report archive index.

STARGATE is a recall surface only. Canonical reports remain in their source repos:

- PriceSEER reports stay under `PriceSEER/`
- Animal reports stay under `PriceLIVE/Animal/`
- PriceMESH reports stay under `PriceMESH/`
- ORG reports stay under `agent-org/`

The archive index lets the Stargate terminal show user-recall entries without becoming the modelling, telemetry, or execution authority.
